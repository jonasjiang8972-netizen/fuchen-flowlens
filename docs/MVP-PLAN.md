## 七、MVP 验收清单

### P0 需求验收（18条中 MVP 覆盖 8条）

| 编号 | 需求 | 验收标准 | 验证方式 |
|------|------|----------|----------|
| FR-AST-001 | API自动发现 | 100个已知API发现率≥98% | 回放测试 |
| FR-AST-002 | 影子/僵尸识别 | 准确率≥95% | 人工核验 |
| FR-AST-003 | 资产分组认领 | 3级分组+30天升级 | 功能测试 |
| FR-DET-001 | BOLA检出 | 模拟遍历1000个ID检出率≥90% | 红队测试 |
| FR-DET-002 | 认证失效 | 单IP 5分钟50账号检出≥95% | 回放测试 |
| FR-DET-005 | BFLA检出 | 管理端点越权检出≥95% | 红队测试 |
| FR-ALT-001 | 风险评分 | 高风险确认真风险≥85% | 人工复核 |
| FR-ALT-002 | 攻击溯源 | 前24小时行为全覆盖 | 功能测试 |

### 非功能验收

| 指标 | 目标 | 验证方式 |
|------|------|----------|
| 单节点 QPS | ≥ 10万 | wrk/自研压测 |
| 检测时延 P95 | < 200ms | Flink metrics |
| 告警生成时延 | ≤ 5s | 端到端测试 |
| Agent 心跳检测 | 60s 内告警 | 断网模拟 |
| 资产增量发现 | ≤ 5min | 注入新流量验证 |

## 八、V1.0（MVP 后 6 个月）预告

MVP 验证通过后，V1.0 将补齐：

- **检测引擎**：剩余 OWASP API Top 10（FR-DET-003~010）
- **敏感数据**：FR-DLP-001~004（分类分级、流动图谱、脱敏核验、合规报告）
- **SOAR联动**：FR-ALT-003（联动处置）
- **业务风控**：FR-RISK-001~003
- **增强前端**：API画像、自定义报表
- **系统管理**：SSO + OAuth
